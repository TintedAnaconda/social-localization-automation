\# Social Localization Automation



\## Prerequisites (install once)



1. Before running setup, install:



\- Git

\- Python 3.11

\- Docker Desktop



Make sure Docker Desktop is running before starting the workflow.



2\. Clone repo



3\. Run setup:

&#x20;  powershell -ExecutionPolicy Bypass -File .\\scripts\\setup.ps1



\---



\## Start workflow



1\. Start Docker Desktop

2\. Run:

&#x20;  .\\scripts\\start.ps1

3\. Open:

&#x20;  http://localhost:5678



\---



\## Stop workflow



.\\scripts\\stop.ps1
